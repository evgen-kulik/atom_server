В этом репозитории находятся следующие скрипты GTAV для платформы FiveM.

ch_spawn - возрождение в заданых координатах и в заданом скине;
ch_car - вызов транспорта;
ch_teleporter - телепортация одного игрока к другому;
ch_hoodcontrol - открытие/закрытие багажника/капота с/без применением потоковости;
ch_gui - отправка простых уведомлений, отправка расширеных уведомлений, оповещения (Alerts), маркер игрока, субтиртры, ввод уведомления со счетчиком)
ch_greeting - вывод приветствия при старте игры, формирование приветствия, сохранение приветствия в базе данных MariaDB.
